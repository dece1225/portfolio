세부 소개

※ 교육 과정

코리아IT아카데미 <정보보안 & 웹서버 개발> 과정 이수

  • 웹서버 프로그래밍 - Java, JSP, Servlet
  • DB 활용 - Oracle, MySQL
  • 웹/서버 보안 - SQLi, XSS, CSRF 대응
  • 리눅스 서버 - 웹서버 구축 및 보안 설정
  • 네트워크 인프라 - 서비스(DNS, FTP, TFTP) 구축
  • 팀 프로젝트 - 기업 맞춤 보안 솔루션 설계

※ 팀 프로젝트

  • 중간 프로젝트 - 함께걸음(커뮤니티 웹 서비스 구축)

    — 역할 : 관리자 페이지(회원관리, 공지사항관리, 문의사항관리), 서초구 지하철 정보 페이지

    — 성과 : DB 실시간 연동(100%), 비동기 처리로 성능 개선(80%)

  • 세미 프로젝트 - 보건제공재단(네트워크 및 인프라 구축)

    — 역할 : TFTP, DNS, FTP 설정

    — 성과 : 설정 백업(100%) 및 파일 전송 환경 구축(80%)

  • 최종 프로젝트 - 불펜토크(보안 점검)

    — 역할 : 윈도우 서버·보안장비 구축

    — 성과 : DHCP(100%), AD(90%), SFTP(90%), UTM 운용(100%)
