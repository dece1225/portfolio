중간 웹개발 프로젝트

노션 포트폴리오

세미 인프라 구축 프로젝트

- 패킷 트레이서, GNS3 활용한 네트워크 인프라 구축
- 리눅스 서버와 연동하여 서비스(DNS, FTP, TFTP 등) 제공

노션 포트폴리오

최종 인프라 구축 및 보안 점검 프로젝트

- 기업 환경에 적용 가능한 보안 솔루션 설계 및 구축

- KISA 제공 취약점 진단 이용한 보안 점검

노션 포트폴리오

소개

경력 하이라이트

  • 중간 프로젝트 - 함께걸음(커뮤니티 웹 서비스 구축)

    — 역할 : 관리자 페이지(회원관리 메인/상세 페이지, 공지사항관리 메인/상세 페이지, 문의사항관리 메인/상세 페이지), 서초구 지하철 정보 메인/상세 페이지

    — 성과 : 웹 페이지에 실제 DB값 실시간으로 연동, 비동기 처리 통한 네트워크 요청 효율적 처리

  • 세미 프로젝트 - 보건제공재단(네트워크 및 인프라 구축)

    — 역할 : 서비스 설정(TFTP, DNS, FTP)

    — 성과 : TFTP 통한 설정 백업, DNS 서비스 구축, FTP 파일 전송 서비스 구축

  • 최종 프로젝트 - 불펜토크(인프라 취약점 진단 및 보안점검)

    — 역할 : 윈도우 서버와 보안장비의 구축 및 보안 점검, ISMS-P 현황 및 가이드 제공

    — 성과 : 윈도우 DHCP, SFTP, AD, Patch 서버 구축, UTM 보안 장비 운용, ISMS-P에 대한 이해

교육 과정

코리아IT아카데미의 <(정보보안 & 웹서버 개발) 웹 보안 솔루션 구축 융합과정> 이수

  • 웹서버 프로그래밍
    - 웹 개발 기초 및 서버 사이드 프로그래밍(Java, JSP, Servlet)
    - Oracle 데이터베이스 연동 및 CRUD 구현
  • 데이터베이스 이해 및 활용
    - 관계형 데이터베이스 개념 및 설계
    - SQL 문법(DML, DDL, DCL) 실습 및 성능 최적화
  • 웹 보안 및 서버 보안
    - SQL 인젝션, XSS, CSRF 등의 보안 위협 대응
    - 보안 강화를 위한 보안 패치 및 웹 방화벽 설정
  • 리눅스 및 서버 관리
    - 리눅스 환경 설정 및 웹서버 구축
    - 서버 운영체제 보안 및 접근 제어
  • 중간 프로젝트 (Java 기반 웹 서비스 개발)
    - Java/MVC2 패턴을 활용한 실전 프로젝트 개발
  • 세미 프로젝트 (기초 네트워크 인프라 구축)
    - 패킷 트레이서, GNS3 활용한 네트워크 인프라 구축
    - 리눅스 서버와 연동하여 서비스(DNS, FTP, TFTP 등) 제공
  • 최종 프로젝트 (기업 맞춤형 웹 방화벽 구축)
    - 기업 환경에 적용 가능한 보안 솔루션 설계 및 구축
    - 팀별 프로젝트 진행 및 발표
